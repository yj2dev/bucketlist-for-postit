## 개발 인원
- 개인
## 실행 방법
1. 해당 레포지토리 클론
2. 커맨드창에 필요 패키지 설치 및 프로젝트 실행 명령어 입력
```
npm i
npm start
```
## 구현 내용
### 1. 버릿리스트 조회 및 생성 페이지
- 버킷리스트 목록을 메모지 형태로 조회
- 메모장 색은 랜덤으로 지정해 항목간 구분 편의 고려
- 완료된 버킷은 배경에 남기거나 복구 가능
- 버킷 삭제 및 삭제 확인 모달 구현
- 버킷 생성 폼 디자인 및 구현
### 2. 버킷리스트 수정 페이지
- 제목 및 내용 수정
